# 🎓 Aprendendo Programação em C

Repositório dedicado aos meus estudos de programação em linguagem C durante a faculdade de Engenharia de Software.

## 📖 Sobre

Este repositório documenta minha jornada de aprendizado em C, desde o básico até projetos mais complexos. Aqui você encontrará exercícios, desafios e projetos desenvolvidos ao longo do curso.

## 📚 Conteúdo

### Fundamentos
- **Hello World** - Primeiro programa
- **Variáveis e Tipos de Dados** - int, float, char, double
- **Entrada e Saída** - printf, scanf
- **Operadores** - Aritméticos, relacionais, lógicos

### Estruturas de Controle
- **Condicionais** - if, else, switch
- **Laços de Repetição** - for, while, do-while

### Funções
- **Funções básicas** - Declaração, parâmetros, retorno
- **Recursividade** - Fatorial, Fibonacci

### Ponteiros e Memória
- **Ponteiros** - Conceitos básicos
- **Alocação Dinâmica** - malloc, calloc, free
- **Arrays e Ponteiros** - Relação entre eles

### Estruturas de Dados
- **Arrays** - Unidimensionais e multidimensionais
- **Strings** - Manipulação de texto
- **Structs** - Estruturas personalizadas
- **Listas Encadeadas** - Implementação básica

### Arquivos
- **Manipulação de Arquivos** - fopen, fwrite, fread, fclose

```bash

gcc nome_arquivo.c -o nome_programa

# Executar
./nome_programa
```
